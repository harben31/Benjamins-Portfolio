What Song is That   the app

About What Song is That the app

This application incorporates the Spotify and MusixMatch APIs. The user can input lyrics into the search and they if the song is in the MusixMatch database they will be returned several cards with the artist, song and a link to listen on Spotify. As this is an entirely client side app oAuth2's implicit grant flow is used to get permissions from the user's Spotify account.

About Workday Calender

Users can input meetings, deadlines or other data onto the hour block. The blocks are color coded gray for past hours, red for current hour and green for future hours. The data can be saved into localStorage if the user clicks the save button on the right side of the hour black and will be retrieved until the users clicks the clear schedule button.
